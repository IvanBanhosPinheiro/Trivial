Trivial

Este proyecto es mi trabajo final para la asignatura de Diseño de Interfaces. Consiste en una aplicación que se conecta a una API para gestionar un juego de trivial con veinte preguntas, registrando las respuestas del usuario y calculando una puntuación final.

La API ha sido desarrollada en Spring como parte de mi trabajo final en la asignatura de Acceso a Datos, integrando así ambos proyectos en un único sistema funcional.

Soy Iván Baños Piñeiro, estudiante del ciclo superior de Desarrollo de Aplicaciones Multiplataforma en el instituto de Maceda. Espero que este proyecto refleje mi esfuerzo y aprendizaje a lo largo del curso… ¡y que consiga un buen aprobado!
